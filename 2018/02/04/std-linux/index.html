linux基础

shell kernel

CLI: Command Line Interface | GUI: Graphical User Interface

提示符: # => root user | $ => 普通 user

命令:

  • -a,–all &后台运行进程 ls
  • ctrl+c 终止当前命令
  • clear 清屏
  • tab 自动补全
  • 上下键或history 历史命令
  • !! 重复前一个命令
  • !c 重复前一个以c开头的命令
  • !n 以历史记录序号执行
  • !?abc 重复之前包含”abc”的命令
  • !-n 重复n个命令以前的命令
  • ctrl+r 历史命令的搜索功能
  • esc 调用上个命令的参数

    用户命令:

  • su - username 切换到root用户
  • sudo 使用管理员权限
  • id 显示用户信息
  • passwd 修改当前用户信息

    进程命令:

  • jobs 查看这在进行的进程
  • bg[n] 继续后台n号进程
  • fg[n] 前台运行n号进程

linux文件系统结构

倒转的单根树状结构，根为”/“,大小写敏感。
每一个shell或进程都有一个当前工作目录，pwd命令显示当前工作目录。
touch命令创建一个空白的文件，或更新已有文件的时间。
以”.”开头的文件为隐藏文件。
命令:

  • ls -a 显示所有文件
  • ls -l 显示详细信息
  • ls -R 递归显示子目录
  • ls -ld +目录或链接 显示目录或链接信息
  • file 查看文件类型
  • cd 切换目录
  • “..” 上级目录 “.”当前目录 “~”家目录 “_”上一个工作目录
  • cp 源文件 目标文件 复制文件或者目录
  • cp -r 复制文件树 -v 显示复制过程详细信息
  • mv 源文件 目标目录/文件名（可选） 移动
  • rm 删除文件，目录 -r 递归删除目录中的所有内容 -i交互式删除 -f强制删除，无提示
  • mkdir 创建一个目录 rmdir 删除一个空目录
  • cat 查看文件内容

    linux文件系统

  • bin:(binary)保存可执行二进制文件， 例如命令
  • boot:引导目录，内有内核，uname,与系统的启动有关
  • dev:(device)硬件设备，被抽象为文件
  • etc:配置文件.conf
  • lib:(library)库文件
  • media:mnt:挂载用，自动挂载
  • opt:大型软件
  • proc:内存中的目录，显示系统的实时信息，内存
  • sbin:(super binary) 只有root用户可以执行
  • sys:底层目录
  • tmp:临时目录
  • usr:应用软件的默认位置
  • var:经常变的文件，

    常用命令

    日期和时间

  • data -U UTC时间 +%y–%m–%d
  • data -s 修改时间
  • hwclok(clock) 显示硬件时钟
  • cal 查看日历
  • uptime 查看系统运行时间

    输出查看

  • echo 显示输入的内容
  • cat 显示文档内容
  • more 翻页显示文件内容(只能向下翻页)
  • less 可以上下翻页
  • head -n 显示文件的开头n行，默认显示10行
  • tail(尾巴) 显示文件末尾几行 -f 追踪显示文件的更新，一般用于查看日志

    查看硬件信息

  • lspci 查看PCI设备 -v详细信息
  • lsusb 查看usb设备
  • lsmod 查看可加载模块(即驱动）

    关机，重启

  • shutdown -h 关机 -r 重启 now立即执行 +10十分钟后执行 23:30定时执行
  • poweroff 立即关机 reboot 立即重启

    归档，压缩

  • zip 压缩后的文件名 文件
  • unzip 解压
  • gzip 压缩
  • tar -cvf out.tar linuxcast 归档
  • tar -xvf linuxcast.tar 解归档
  • tar -cvzf 归档并压缩

    查找

  • local keyword 快速查找文件 需要建立数据库 updatedb手动更新数据库
  • find 查找位置 查找参数 -name按名称查找 -perm权限 -type

VI&VIM

vi是命令行界面下的文本编译器 vim更强
使用命令vi或者vim 在命令行中启用vim

  • vim 目标文件 如果目标文件存在，则打开。不存在，则创建。

    esc:

  • i 插入模式
  • o 在当前行下插入新行
  • dd 删除整行
  • yy 复制整行 n+yy 多行复制
  • p 粘贴
  • r 替换当前字符
  • / 查找关键字

    ex:

  • w 保存当前修改
  • q 退出，q!强制退出
  • x 保存并退出 = wq
  • set number 显示行号
  • !系统命令 执行一个系统命令
  • sh 切回命令行 ctrl+d 切回vim
